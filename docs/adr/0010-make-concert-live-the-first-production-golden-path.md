# Make Concert Live the first Production Golden Path

Roy AI Editor 先把 Concert Live Workflow 完成為第一條 Production Golden Path，再以 Directed Footage Workflow 作為第二條；Reels／Shorts 優先成為核准長片或素材庫的衍生輸出，Talking-head Localization 再評估 BaoCut／VideoLingo adapters，旅遊、機器人比賽與生成式 B-roll 等專門 Workflow 後續組合已驗證能力。第一個正式里程碑從 YouTube URL 開始，涵蓋權利與來源證據、曲目與切點、歌詞／翻譯研究及核准、對軸、字幕、渲染、獨立 QA 與 private YouTube 上傳，最後回傳連結；Concert V1 Acceptance 要求三個 Production Capability Spikes 與 Concert Golden Corpus 全部通過，並完成單人／無來源日文、多人／完整來源日文含 Safe-Area Recovery、cover／翻譯與讀音困難三類真實 Media Projects，合計至少 12 個 Track Jobs。驗收還必須包含 Mac 重啟續跑、stage retry、重複 URL 與 upload retry idempotency、Publish Verification、Retention Plan dry-run、Recovery Vault backup／restore drill，且不依賴人工修改 JSON、搬檔、臨時腳本或必要 GUI 操作；正常路徑只有一次 Concert Lyrics Review，其他歌曲級問題最多集中成一次 Exception Digest。通用能力只從這條已驗證路徑中抽取，Directed Footage 與其他影片類型可以研究介面但不得搶占 Concert V1，unlisted／public Visibility Promotion 仍是獨立決策。
