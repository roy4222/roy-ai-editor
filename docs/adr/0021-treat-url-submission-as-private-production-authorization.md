# Treat URL submission as private production authorization

Roy 主動提交 Concert URL 時即建立 Intake Authorization，允許 Roy AI Editor 下載來源、在本機製作並把通過 QA 的 Verified Render 上傳為 private，不再為每個正常來源要求固定的 `approve-rights`。流程仍須執行 Rights Audit：官方規範明確允許時自動繼續，明確禁止或需要繞過登入、付費牆、DRM 等存取控制時停止，找不到規範或證據矛盾時才進入 Exception Review。這項授權不表示 Roy 聲稱擁有來源著作權，也不包含 unlisted／public Visibility Promotion。
