# Base Roy AI Editor on video-autopilot-kit

Roy AI Editor 將以 `Hao0321/video-autopilot-kit` 作為上游基礎，再加入 Roy 專屬的 profiles、能力、工作流程與品質標準；不再把兩者視為獨立產品與被動參考快照。這讓專案直接承接 upstream 已有的影片製作、QA、模板與知識骨架，同時把九支 Live 成片累積的做法轉成可重用、可版控、可持續改進的 Roy Customization。Git 血統與同步方式由 [ADR 0002](./0002-maintain-upstream-git-lineage.md) 決定。
