# Keep the public Repo free of private and restricted data

Roy AI Editor 維持公開 Repo，並版控程式碼、Skills、通用 workflows、Public Customization、schemas、文件與安全的合成 Test Fixtures。Credentials、私人聯絡資訊、未公開分析資料與其他 Private Configuration 不進公開 Git；真實影片、受授權限制的歌詞／翻譯及未公開素材則依 Production Asset 邊界留在 D 槽。公開 Repo 為私密欄位提供 schema 或去識別範例，push 前必須檢查秘密、個資、授權內容與大型媒體。
