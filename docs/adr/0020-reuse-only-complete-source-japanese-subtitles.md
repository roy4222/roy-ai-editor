# Reuse only complete source Japanese subtitles

Concert Live 在 Source Subtitle Audit 證明日文字幕覆蓋 100% performed lines、與核准 Lyrics Packet 的文字／順序／時間一致，且下方有安全繁中區域時，才進入 Source Japanese Subtitle Mode：沿用來源日文並只新增其下方繁中。任一缺行、錯字、時間不符或空間不足都將該 Track Job 送進 Exception Review，不以逐行混搭樣式或雙重日文換取表面上的自動化。
