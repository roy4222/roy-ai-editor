# Build Directed Footage as the second Production Golden Path

Concert V1 Acceptance 完成後，Roy AI Editor 的第二條 Production Golden Path 是 Directed Footage Workflow：接收本機／Google Drive 素材集合與 Roy 的自然語言 Editing Brief，自動建立 Asset Inventory、proxy、內容理解、結構化 Edit Plan／EDL、render、QA 與 private link。Reels／Shorts 優先作為核准長片或同一素材庫的衍生 Workflow，避免先做與通用素材理解分叉的一鍵短片工具；Talking-head Localization 其後再評估 BaoCut／VideoLingo adapters，旅遊、機器人比賽與生成式 B-roll 等專門 Workflow 從前兩條已驗證路徑組合能力。Concert V1 前可以研究這些介面與 boundaries，但不得分散其實作優先級。
