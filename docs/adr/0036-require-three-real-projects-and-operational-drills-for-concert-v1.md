# Require three real projects and operational drills for Concert V1

Concert Live 只有達成 Concert V1 Acceptance 才能宣告為第一條 Production Golden Path：三個 Production Capability Spikes 與完整 Concert Golden Corpus 必須全綠，並在單人／無來源日文、多人／完整來源日文含 Safe-Area Recovery、cover／翻譯主軸與讀音困難三類真實 Media Projects 上完成至少 12 個 Track Jobs。每個正常 project 從 URL 單獨啟動到 private links，不得要求人工改 JSON、搬檔、補 shell 或操作必要剪輯 GUI，固定人工 gate 只有一次 Concert Lyrics Review，歌曲級例外最多集中成一次 Exception Digest。驗收必須實際演練 Mac 重啟續跑、stage retry、重複 URL、YouTube upload retry、Publish Verification、30 天 Retention Plan dry-run、Recovery Vault backup 與 restore；任何不能自動完成的情境都須安全隔離、留下完整 evidence 且不靜默產出錯片。在此之前其他 Editing Workflows 可以研究，但不得搶占 Concert V1 實作優先級。
