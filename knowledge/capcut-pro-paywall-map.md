# CapCut Pro Paywall Map (M33)

> ⚠️ **若你已訂閱 CapCut Pro** → **所有 💎 模板 / 特效 / 濾鏡 / 轉場 全部解鎖**
> M33 paywall 規則此時**不再 enforce**。此檔保留作為**未訂閱 fallback 知識**。
> 取消訂閱時恢復套用本檔規則。

## ⚠️ 訂閱中的 status

**Agent 可自由套用 💎 Pro templates / effects** — Export 不會被 block。
但仍要避開：
- 「智能 X / AI X / 自動 X」可能仍有額外限制（驗證再用）
- 4K Export 可能需確認 plan tier
- 部分 cloud features（AI Avatar / 文本朗讀）可能需 extra credit

## 以下為 Free 版規則（取消 Pro 時恢復）

## 識別 Pro 圖示

| Icon | 意義 |
|---|---|
| ⬇ **下載 icon**（圓圈內箭頭向下） | **免費** — 第一次點會下載 |
| 💎 **菱形 icon**（紫/橘色寶石） | **Pro 鎖** — Export 時跳升級警告 |
| 「智能」/「AI」/「自動」字眼 | 80% 機率是 Pro，不要點 |

## 已確認 Pro 鎖功能

### Transition (轉場) — 大部分 Pro
- ❌「春日光效」(spring light effect) — 菱形 icon
- ❌「電光特效」
- ❌「氛圍光效」
- ✅ 「基礎」 → 「淡入淡出」「黑場」「白場」是免費

### Effects (特效)
- ❌「自動調整」 / Auto Color Adjustment — Pro
- ❌ 大部分「藝術」「光影」類菱形 icon

### Filters (濾鏡)
- ❌ Pro 濾鏡多數菱形
- ✅ 「基礎」 → 「明亮」「暗調」少數免費

### 文字模板 (Text Templates)
- ✅ Dynamic white crayon hand-drawn (確認 free，agent 跑過)
- ✅ Colorful Paint Graffiti (確認 free，但偏兒童風)
- ⚠️ 「綜藝」分類**大部分**是 Pro，挑時看 icon
- ⚠️ 「藝術字」分類大部分 Pro

### Stickers (貼圖)
- ✅ 「基礎」分類大部分 free
- ❌「動態貼圖」大部分 Pro

### 其他確認 Pro
- ❌「智能剪輯」(AI Auto Edit)
- ❌「智能字幕」(AI Caption)
- ❌「智能美顏」
- ❌「智能摳圖」
- ❌「文本朗讀」(TTS)
- ❌ 4K 解析度匯出
