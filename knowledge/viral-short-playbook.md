# 爆款短影片剪輯 Playbook（從頭部教學頻道反推）

> 來自 video-autopilot-kit 開源知識庫 · MIT 授權

> 反推自某頭部教學頻道（示意）的一支高互動課程促銷 short。
> 方法論：對一支高表現短影片做逐刀 (cut-by-cut) 拆解，提煉可複用的剪輯技法。

---

## 🚨 INTEGRATE / CALIBRATE / REPLACE Matrix（最重要規則）

**這是套用任何 reference playbook 的最重要規則**。Reference 是一個 pattern library，**不是** personality clone。把別人的爆款拆出來的技法，要分成 3 類處理，避免把參考對象的人格特徵硬套到自己身上：

| 類別 | 技法 | 用法 |
|---|---|---|
| ✅ **INTEGRATE**（universal craft，無人格衝突） | **A1** 14 秒一鏡 / **A2** 慢→快→慢弧 / **A3** cut on caption / **B3** multi-pane / **B4** data viz / **C2** editing UI / **C3** thumbnail wall / **C5** Pro tool close-up / **D2** cut on beat / **D3** AI voice mock / **D4** SFX / **E1-E5** promise | 直接套，不用 calibrate |
| ⚙️ **CALIBRATE**（依自己舒適區調整程度） | **B1** slogan card → 用自己的色彩 palette（不必照抄純黑底）/ **C1** NAMING SELF → 用自己的語氣（硬尾 vs 軟尾）/ **D1** LUFS → 依品牌調性微調（不一定推到參考對象的極端值）/ **G** 極端化 → thumbnail 可極端，但 audio/cut 程度依自己節奏 | 套技法但程度依既有節奏微調 |
| ❌ **REPLACE — 永遠用自己的 signature** | **F1** silhouette / sunset → 用**你自己的品牌 outro 卡**（若你的頻道不露臉，就用無人入鏡的字卡）/ **F2** hand-on-chin pose → 若你的頻道不露臉，改用 outro 字卡軟尾招牌（用你自己的結尾招牌句）/ **B2+C4** phone view count screenshot → 換成**你自己的社群截圖 / 你自己的數據** | 兩種人格互斥，永遠保留你自己的 signature |

**永遠保留（無參考對照）**：你自己的品牌 outro / 訂閱提示 / 你的社群 CTA。

**為什麼這個 matrix 重要**：常見錯誤是把整份參考 playbook 寫成 MANDATORY default → 強迫自己錄參考對象的 silhouette / hand-on-chin pose（F1/F2）→ 完全違反自己既有的 signature。教訓：**Reference = pattern library，不是 personality clone。Integrate 不 replace 你自己。**

## 🎯 為什麼學頭部教學頻道

- 某頭部教學頻道（示意）— 訂閱量級在同類教學頻道中名列前段
- 內容**極端化**先驅：據其公開說法，封面被迫越做越誇張，否則難以吸引點擊
- 自述工作流：**先 AI 旁白抓節奏 → 剪輯師依節奏剪 → 創作者補正式旁白**
- 據其公開說法，平台已演變成比誰音量更響的競賽 — 音量響度比十年前高好幾倍

→ 爆款公式 = **「節奏 + 視覺極端化 + 權威堆疊 + 情感收尾」** 4 維度同時拉滿

## 📐 樣本影片骨架（約 1:52 = 22 個 cut）

| Beat | t | 內容 | 視覺 | 字幕 |
|---|---|---|---|---|
| **Hook** | 0-14s | 自我介紹 | 一鏡到底 talking head（**14 秒不切**）| 「我創立了這個帳號」|
| **Authority A** | 14-18s | 露出剪輯軟體 | 背身 + 剪輯 timeline | 「剪輯」字 |
| **Authority B** | 18-22s | 過往作品 wall | 全螢幕縮圖牆快切 | 「我們做過許多專業的剪輯課程」|
| **Curriculum** | 22-35s | 課表 reveal | 列表 + 模組標題 | numbered list |
| **Promise** | 35-43s | 零基礎承諾 | 多軟體 UI 並排 + 價格 banner | 「我們會從零基礎開始」|
| **Pillar 大字卡** | 43-48s | 三大支柱 | **黑底白字「文字 / 畫面 / 聲音」** | 大字 own frame |
| **Data Vis** | 43-47s | 課程結構 | bars chart「持續時間」+ 多機位剪輯 | — |
| **WORKFLOW Promise** | 48-65s | 不只教功能 | 手機協作板 + 桌機 | 「我們不只教你功能」 |
| **Pro Authority** | 65-78s | 調色面板 | 色輪 close-up | 「全能必修課」|
| **Social Proof** | 78-100s | 爆款證明 | phone 顯示高觀看數 + 相機裝備 | 「甚至我們之前那些爆火的影片」|
| **Emotional CTA** | 100-108s | 浪漫剪影 | 情侶看海 + timeline strip | 「如果你想從零開始學剪輯」|
| **Price + Pose** | 100-112s | 低價 anchor + 反思手勢 | 紅字價格 + 手托下巴 | — |

### 關鍵節奏觀察

- **第一刀在 t=14s** — 前 14 秒**一鏡到底**鎖住觀眾（反直覺：短影片 ≠ 一開始就快切）
- **t=14-22s 8 秒 6 刀** — Authority stack 突然快起來（**節奏對比** = 喚醒）
- **後 90s 平均 4.5 sec/cut** — 穩定快節奏
- **結尾再放慢**（情侶 silhouette 停留 ~3 sec）— 情感收束 vs 開頭一鏡呼應

## 🛠️ 7 大爆款剪輯技法（可偷可套）

### A. 節奏 (Pacing)

| 技法 | 怎麼用 | 應用範例 |
|---|---|---|
| **A1 — 反直覺 14 秒一鏡** | 開頭不切片，臉 + 聲先建立信任 | 教學長片開頭 5-10 sec 不切 |
| **A2 — 慢→快→慢三段弧** | hook 慢、demo 快、CTA 慢 | 教學影片用：introduction 慢 / demo 快切 / 結尾慢 |
| **A3 — Cut on caption change** | 字幕一變就切畫面 | dual-tier 字幕 + 每張對應一個 b-roll cut |
| **A4 — AI voice 先抓節奏** | 剪輯師依 AI 旁白剪 → 真人補錄 | 用 AI 語音先 mock 旁白給剪輯參考 |

### B. 視覺層疊 (Visual Layering)

| 技法 | 怎麼用 | 應用範例 |
|---|---|---|
| **B1 — 大字 Slogan Card** | 黑底 + 30%+ 螢幕高白字 own frame | 教學 chapter 過場：「文字 / 畫面 / 聲音」式 3-stack |
| **B2 — Phone Mockup 內嵌** | 影片中放手機展示另一影片 / 截圖 | 秀某個按鈕或社群訊息可用此技法 |
| **B3 — Multi-pane 同時 reveal** | 3-4 panels 並排顯示課表 | 教學長片秀「教學大綱 / 工具表 / 流程圖」三併排 |
| **B4 — Data Vis 證據化** | bars chart 顯示「持續時間」量化 | 秀數據時用 bars 不用 plain text |

### C. 權威堆疊 (Authority Stacking)

| 技法 | 怎麼用 | 應用範例 |
|---|---|---|
| **C1 — Open by NAMING SELF + 帳號** | 「我創立了 X 這個帳號」3 秒內 | 開頭：「我是 ___ 工作室創辦人」|
| **C2 — Editing UI B-roll 15s 內出現** | 剪輯軟體 timeline 入鏡 | 教學影片開頭丟 1 sec 剪輯 timeline B-roll |
| **C3 — 過去作品 thumbnail wall** | 全螢幕縮圖快切 | 秀過去多支作品 thumbnail wall |
| **C4 — Phone 顯示 view count** | 手機 mockup 顯示爆款數字 | 秀某支影片 24h 數據用 phone mockup |
| **C5 — Pro tool close-up** | 色輪 / curve / 高階面板 close-up | 秀 3D 軟體節點 / 材質貼圖細節 |

### D. 聲音設計 (Sound Design — 創作者自述核心)

| 技法 | 怎麼用 | 應用範例 |
|---|---|---|
| **D1 — 響度比同行高** | LUFS 推到 -10 ~ -8（YouTube cap 是 -14）| post-process 把 master 推到約 -10 ~ -12 LUFS（依品牌調性，不必到極端） |
| **D2 — Cut on beat** | 音樂 BPM 對齊 cut 點 | BGM 找有明顯 bass kick，每 kick = 1 cut |
| **D3 — AI voice 先 mock 節奏** | AI 語音出 draft → 剪片依此節奏 | 教學長片可省 voice 重錄成本 |
| **D4 — Big-text card 配 SFX** | 大字進場配 whoosh / hit sound | 剪輯軟體自帶 sfx 庫即可 |

### E. 課程 / Promise 結構

| 技法 | 怎麼用 | 應用範例 |
|---|---|---|
| **E1 — 「從零基礎開始」貼紙** | 降低門檻 | 教學：「我不會寫 Code 也能...」 |
| **E2 — 3 Pillar 大字** | 文字 / 畫面 / 聲音 三柱 | 教學：「Prompt / Tools / Output」3 柱 |
| **E3 — 「百集」「完整」volume signal** | 高數字營造完整感 | 「N 款作品 / N 天 / N+ prompts」 |
| **E4 — Price anchor 低** | 低價 vs 一般課程行情價 | 賣任何產品可用此 anchor |
| **E5 — 「不只教功能還教 workflow」** | workflow > tool 教學定位 | workflow 教學定位比單純 tool 教學更有差異化 |

### F. 情感收束 (Emotional CTA)

| 技法 | 怎麼用 | 應用範例 |
|---|---|---|
| **F1 — Silhouette + sunset / 海** | 情侶剪影看海 = 「你能創造這種回憶」 | 旅遊 vlog 結尾可含；教學影片可用「你也能做出像這樣的東西」b-roll（若你的頻道不露臉則用無人空景） |
| **F2 — Hand on chin 反思 pose** | 軟化 sales pitch，feels personal | 教學結尾換個 pose 講「下一支會公開...」（若你的頻道不露臉則用 outro 字卡軟尾 + 你自己的結尾招牌句） |
| **F3 — Sound fade out** | 結尾音量緩降不 hard cut | 剪輯軟體 audio fade 1-2 sec 必加 |

### G. Meta — 極端化（間接整理自公開說法）

> 該頻道公開談過：封面被迫越做越誇張，否則沒人點進來，連聲音也成了戰場。
> 也提到十年前的影片音量明顯較低，如今大家都把音樂拉大聲，平台演變成比誰更響的競賽。

→ **短影片紅線**：
1. Thumbnail 不誇張 = 死
2. Audio 不大聲 = 滑掉
3. Cut 不快 = 退出
4. Caption 字不大 = 看不到

→ 應對：
- Thumbnail 走 MrBeast 反差
- LUFS 推高（D1，依品牌調性）
- Cut 4.5 sec/avg（A2）
- Caption 30%+ 螢幕高（dual-tier 字幕）

## 🔗 延伸原則

- hi-res frame 配畫面：每張字幕對應對得上內容的畫面，避免「講 A 卻秀 B」的 mismatch
- 雙 tier 字幕樣式：對齊 caption-driven pacing 風格
- agent GUI hygiene：用 agent 剪 short 前的環境清理（隱私 / chrome / 視窗）
- Pacing 原則：A1-A4 節奏技法是 editing 的核心一層
