### 1. **Hype button 制度化**（最高 ROI — beta 國 unfair advantage）

部分地區是 **YouTube Hype button beta 國**（全球僅少數國家）。中小頻道在此是 sweet spot：
- **小頻道 multiplier**（500–500K subs 合格）
- **既有鐵粉社群可動員**（11 天可見 +95% growth 級別的動員效果）
- 每人每週 3 hypes × 社群人數 = 可觀的週點數
- 上 Country leaderboard = Explore 額外曝光 = 繞過 Dec 2025 Browse -80% 的長片砍殺

來源：[YouTube Blog](https://blog.youtube/news-and-events/youtube-hype/) | [OutlierKit](https://outlierkit.com/resources/youtube-hype-feature-small-creators-growth-2026/)

### 2. **頻率改成 1 長片 + 3-4 Shorts/週，Vlog 優先轉 Shorts 主力**

Dec 2025 YouTube Browse 長片 slot 從 12→2（**-80%**）是結構性的，每週 1 部撐不過 2026。
- Travel vlog 長片 AVD **YoY -19%**、CTR **-22%**
- Travel Shorts upload **YoY +214%**，已佔 travel impressions 38%
- 一趟旅遊行程 → 拍 5-8 個 60-90s POV moment Shorts + 1 長片總集，ROI 3-5x

**3D/軟體教學長片可維持**（Suggested/Search 沒砍）；**Vlog 必須轉 Shorts 主力**。

來源：[Dataslayer Dec 2025 Browse](https://www.dataslayer.ai/blog/youtubes-december-2025-algorithm-update-browse-feed-cut-long-videos-by-80) | [Travel Vlog Decline](https://www.alibaba.com/product-insights/are-travel-vlogs-declining-on-youtube-or-just-shifting-to-shorts-algorithm-trend-analysis.html)

### 3. **IG Reels：location tag 精準到地標 + caption 前 125 字 keyword-front + hashtag 砍到 5**

純設定改動成本最低，立刻吃 2026 三條新規：
- Mosseri 12/2025 官方 hashtag **上限 5 個**（之前 30）
- 地點 tag 貼文 saves+shares **+12%**；自訂地點本地搜尋 **+40%**
- 雙語 caption **+27% engagement** — 完美對應跨市場（例如本地 + 國際雙語觀眾）

**Reels caption 通用模板**：
```
[本地語言 hook 1 句 — ≤15 字撩 save]
[地點名英文 + activity + 痛點解法 keyword]
[CTA 本地語言 — 驅 share/save 不是 like]
#精準地點tag #niche中文tag1 #niche中文tag2 #活動類型 #你的社群tag
```

來源：[Geotag Saves +12%](https://socialrails.com/social-media-terms/geotag-instagram) | [Caption SEO Bilingual +27%](https://www.truefuturemedia.com/articles/instagram-reach-2026-algorithm-reels-carousels-caption-seo)

## 5. External Traffic Ignition（社群 / 演講）

### A. Quality external traffic = 強正向 signal（沒懲罰）

> 「Niche community 來的（Reddit/Discord/FB Group）generate 高 AVD + 高 engagement + 高訂閱轉換，all of which tell YT 算法 deserve broader distribution」

- 首 24-48h 強 engagement = distribution decider
- 都市傳說「外部流量會被罰」**錯誤** — 罰的是 low-quality buy traffic，不是 community

**vs 現有 playbook**：多數 playbook 完全沒談 external ignition

**Actionable — 標準流程**：
1. **首 24h**：推自有社群（Discord / Line / 社群群組）
2. **24-48h**：觀察 retention（≥60% 1-min OK，<50% kill）
3. **48-72h**：Hype button 推第二波（beta 國加成）

⚠️ **不要分散全推首 1h**：撐 24h 推力比一次性 spike 算法更愛（看到持續 engagement）。

來源：[Fourthwall External Traffic](https://fourthwall.com/blog/secrets-of-the-youtube-algorithm) | [Content Guaranteed](https://www.contentguaranteed.com/?p=760)

### B. Share external 加權倍率

External share 估值 **5-8x of like**，是 satisfaction stack 第 2 強訊號（88 vs Surveys 95）。

**Actionable**：演講場合「掃 QR → 開影片 → 分享到 Threads 給朋友」**這動作每次值上千 likes 的算法權重**。
- 演講結尾**直接演示 share**，不是「請按讚訂閱」
- 演講 / AMA 觀眾分享比直接訂閱算法權重高

## 6. 🆕 2026-07 深研 R15-R25（Test&Compare 判贏改制 / Auto-Dubbing / Shorts 脫鉤 / seed 機制 / 滿意度正式化 / 緊 cluster / Communities / AI carousel / Ask Studio / 30s 門檻 / Hype 全球化）

> 2026-07 六路深研合成的 11 條新規（編號 R15-R25，供跨檔引用）。剪輯側對應的 18 項升級 + skip list → `premium-motion-fx.md`。

**R15 — Test & Compare 判贏 = watch-time-per-impression（Quality CTR）非 CTR，且已支援 3 縮圖變體 + 標題 A/B（2025-12 全球開放）**
- 測「承諾強度」（真值數字 / 工具名 / 結果畫面 三變體）不測風格；每變體需 1k-5k impressions、跑滿 2 週不提早收；Inconclusive 回退首圖 → 第一張放最有把握的。
- **若你的 CTR 已 clear ~8%**：縮圖優化方向 = 期待設定的**準確度**，不是更聳動 — 判贏改制後騙點自動輸（證據：官方機制文件 + 2026-01 一項 127-test 實測，表情變因 +42% CTR / 勝率 82%）。

**R16 — Auto-Dubbing 2026-02 起全 YPP 開放（27 語）**
- 開英文（+ 你的第二大潛在市場語言）可直接破**單一語言池的曝光天花板**（官方數據：啟用者平均 25% watch time 來自非母語；名廚頻道案例 3x views）。
- 發布前逐軌預聽審**技術名詞**（ffmpeg / Claude / API 這類詞常被亂翻，可單軌關掉）；長期把關鍵數據做成「數字 + 圖示」畫面、少依賴燒死的單語硬字幕。

**R17 — Shorts 與長片分發 2025 底正式脫鉤**
- Shorts = 純增量入口，爆不爆**不拖累**長片池。
- 長片發布 72h 內出 2-3 支直式 payoff Shorts、每支掛 related-video link 回長片 + 尾 2-3s 字卡「完整參數在關聯影片」（該 link 點擊 = 2026 高權重轉換訊號）。
- Shorts 訂閱轉換 ≈ 長片入口的 1/3 → 只看「Shorts→長片觀看」funnel 數字，不灌 returning-viewer KPI。

**R18 — Seed 機制實證**
- 小頻道首 48h 拿 100-1,000 seed impressions、評估期持續**數週**（「48h 定生死」被官方打臉）。
- day-0 唯一工作 = 1h 內餵熱 seed 樣本（Communities video post + pin、Shorts 入口、自有社群一則）；**48h 內禁 panic 換縮圖/標題**（污染 T&C 樣本）。
- day-14 決策樹：impressions 停滯 + CTR >7% → 修留存；CTR <5% 才動包裝。
- 若你有一支 breakout（演算法分發佔比極高的那支）= seed→擴圈成功樣本，**複製它的 day-0 動作**。

**R19 — 2026-05 滿意度訊號正式化 + contextual weighting**
- 片長 = **密度能撐滿的最長**（以你自己實測過的甜蜜帶為準；不推翻絕對 AVD 邏輯、禁灌水拉長）。
- 最後一個 payoff 講完 **≤8s 進 outro**（你的 outro 簽名可保留但壓 ≤8s — satisfaction-weighted AVP 對片尾拖沓最敏感）。
- mobile 為主的頻道，字級以手機可讀為準。

**R20 — Browse 2026-02 改緊 audience-cluster 匹配**
- 接下來 5-8 支**鎖死同語義簇**（同一工具/主題的實測 + 復盤）不出簇；出簇 = 每支片重新冷啟動、鎖簇 = 每支繼承前一支的觀眾群預測。
- description 前兩行重複 breakout 核心詞組；end screen 第一格永遠 = 同簇上一支；playlist 只按簇分不按時間。
- 你的 breakout 若演算法分發佔比極高 = 已進對簇的證據，別離開它。

**R21 — Communities 2026-06 擴大開放且貼文直接進訂閱者 Home feed**
- 週 2-3 則節奏機械化：上片日 video post + pin ／ 週中 poll 選題驗證（取代猜題）／ 真值數據透明貼。
- 這是**不出片就能觸發回訪**的唯一官方面。健康基準：留言率 >1%、回訪觀眾 >40%。

**R22 — YouTube 搜尋 AI carousel（2025-06 起測試擴大）會抽 clip 直接答查詢、教學 niche 最曝險**
- 對策 = demo 結果 30-60s 前置（被抽走 = 免費鉤子）、完整參數/步驟散佈在 2-5min 片內（clip 只給 what 不給 how）。
- 章節標題用**完整查詢句式**（「ffmpeg 字幕燒錄參數」非「字幕」）+ 口播明確唸出關鍵詞（AI 抽 clip 靠語音轉錄匹配）。

**R23 — Ask Studio 2026-05 大規模開放**
- day-14 retrospective 固定 3 問寫進你的 Log Outcome：(1) 這支觀眾多少比例看過我其他片（回訪歸因）(2) 最大流失點在講什麼 (3) 我的觀眾還看哪些頻道/主題（= 語義池直接窺探，選題對準該池缺口）。答案記進你的 optimization log。

**R24 — 30s 留存 ≥70-75% = 進 Suggested 的門檻（<60% 幾乎停滯）**
- hook 逐秒制：0-5s 真值結果 cold open → 5-15s 承諾 → 15-30s 給第一步（模板細節 → `premium-motion-fx.md` #8）。
- 發布後 72h 把留存曲線 dip 對回 interrupt 排程表：dip 落在 >90s 無 interrupt 空窗 = 補窗；跨影片同相對位置重複 dip = 結構問題（該處下支排 burst / re-hook）。

**R25 — Hype 已 39 國全球化（本檔 §「Hype button 制度化」的 2026-07 更新版）**
- 有效窗 = 發布 day 1-7、每觀眾每週 3 個免費 hype、頻道越小倍率越高（500 訂級 ≈7,500 分/hype — 小頻道正是甜蜜點）。
- day-0 在 outro 卡 + 置頂留言加 CTA、day-7 機械刪除過期 CTA（寫進 publish checklist）。
- 紀律不變：**只給你最有把握（過 Top 1% filter）的片**，不是支支都推。
